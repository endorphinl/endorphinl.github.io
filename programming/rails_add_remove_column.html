Rails: add or remove column

by hokjoung on 2013
1. rails g migration add_color_to_sauce color:string //not necessary to specify the table name because rails is smart.
2. check appropriate migrate file in /db/migrate with the name 'add_color_to_sauce'
(for removing, might have to manually write in the 'remove_column()' function in migrate file)
3. rake db:migrate //actually add the column or delete the column in db
4. change 'param' in 'sauces_controller' (controller for appropriate table)
5. change appropriately the _form , index , show view. (might have to change add, edit if the two do not use _from) (again, change view of appropriate table)
