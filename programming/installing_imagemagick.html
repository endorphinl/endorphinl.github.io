Installing ImageMagick

by hokjoung on 2014
1. tar -xvzf ImageMagick-6.8.8-1
2. go into unzipped folder then './configure' then 'make' then 'make install'
3. tar -xvzf libpng-1.6.8.tar.gz then do step 2 for this folder
4. type below into terminal

PKG_CONFIG_PATH=/usr/local/lib/pkgconfig
export PKG_CONFIG_PATH
gem install rmagick -v '2.13.2' -- --disable-htmldoc

5. gem install carrierwave
6. add following into Gemfile of rails application

gem 'rmagick'
gem 'carrierwave'
