Android Studio tips

by hokjoung on 2014

- Ctrl+Space to finish autocomplete

- Ctrl+N to open new class

- Alt+F7 to find where a particular class, method or variable is used

- Ctrl-Q to preview documentation on hovered class

- Ctrl-B to get to class, method or variable's declaration

- Shift+F6 to rename classes, methods and variables with corrections in all places used

- The Escape key in any tool window moves the focus to the editor.

- Shift+Escape moves the focus to the editor and also hides the current (or last active) tool window.
