Elasticsearch concepts

by hokjoung on 2015

- review include elastic_review

- elastic_review include elastic_common

- elastic_common has a default reindex_all and reindex

- reindex_all is to create an index with data that is not in the form of elasticsearch index

- reindex is to add one data into existing index

- other search methods are created appropriate to uses

- elasticsearch has an analyzer attached

- the analyzer is a plugin that tokenizes strings in a certain way

- the logic part is all in the search query, just like how relational db works (one's search query determines whether you want a ranking or a grouping in mysql)
