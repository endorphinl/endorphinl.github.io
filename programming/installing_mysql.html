Installing MySQL

by hokjoung on 2014
1. vim /hoktraining/config/database.yml

development:
adapter:
encoding:
database:
username:
password:
host: (make sure this is the centos ip address instead of 127.0.0.1 if you are accessing from other computers)
port: 3306

2. go to http://www.percona.com/doc/percona-server/5.5/installation/yum_repo.html
3. do 'rpm -Uhv....'
4. test if works using 'yum list | grep percona'
5. go to http://www.percona.com/doc/percona-server/5.5/installation.html#using-percona-software-repositories?id=repositories:start
6. yum install Percona-Server-client-55 Percona-Server-server-55 Percona-Server-devel-55
7. it will tell where there are conflicts between percona and mysql
8. use 'yum list mysql' to find it and then 'yum erase mysql'
9. do step 5 again until successfully installed
10. now need to start server using 'service mysql start' in any directory
11. then use shortcut to setup passwords and delete tests using

'/usr/bin/mysql_secure_installation'

12. 'y' then set password -> 'y' for remove anonymous -> 'n' for disallow remote root login because will be using mysql workbench software on windows OS to access mysql db -> 'y' for remove test db -> 'y' for reload privilege tables
13. 'gem install mysql2'
14. go to /hoktraining dir then 'bundle install'
15. add 'gem mysql2' into Gemfile in hoktraining
16. 'mysql --user=root -p' then enter password
17. 'create user 'root'@192.168.0.251 identified by '...';'
18. 'grant all privileges on *.* to 'root'@192.168.0.251 with grant option;'
19. 'create database hoktraining;' 'exit;'
20. 'rake db:migrate'
18. 'grant all privileges on *.* to 'root'@192.168.0.203 with grant option'
19. download mysql workbench on windows OS
20. Database -> Connect to Database 192.168.0.251 / root / now pw
